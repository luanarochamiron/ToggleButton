# Animated Toggle Button with Icon

Um componente de botão de alternância (Toggle Switch) moderno, interativo e minimalista desenvolvido com HTML5, CSS3 e JavaScript. O projeto apresenta um interruptor tátil de grande escala que integra animações fluidas de deslizamento e transições de estado para um ícone visual (cadeado aberto/fechado), ideal para fluxos de segurança, ativação de recursos ou telas de bloqueio.

## Tecnologias Utilizadas

* HTML5: Estruturação semântica do contêiner do botão e inclusão do ícone vetorial de estado.
* CSS3: Estilização completa, uso de bordas totalmente arredondadas (estilo pílula), gerenciamento da paleta de cores roxa e transições de movimento.
* JavaScript: Manipulação do Dom para alternar classes de estado (`active`/`checked`), gerenciar a lógica de clique e disparar a transição do ícone correspondente.

## Funcionalidades e Técnicas Aplicadas

* Animação Deslizante Fluida (Smooth Slider): A esfera interna branca desliza horizontalmente de forma responsiva ao receber o estímulo de clique, utilizando propriedades otimizadas de transição em CSS.
* Feedback Visual Baseado em Ícones: Inclusão de um ícone de cadeado centralizado na esfera deslizante que altera dinamicamente seu estado visual para representar as condições de "ativado/bloqueado" ou "desativado/desbloqueado".
* Estética Minimalista e Moderna: Uso de cantos perfeitamente arredondados (`border-radius: 100px`) tanto no trilho quanto na esfera interna, criando um elemento com aspecto de interface de sistema operacional (OS).
* Centralização Absoluta na Viewport: Uso de CSS Flexbox para alinhar horizontal e verticalmente o componente no centro exato da tela branca, eliminando distrações periféricas.
